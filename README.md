## How to Download One Template Only

### Option 1: Download the full repository (Recommended for beginners)
1. Click **Code**
2. Select **Download ZIP**
3. Extract the ZIP file
4. Open the required template folder (e.g. `template-3`)
5. Open `index.html` in your browser

---

### Option 2: Download a specific template manually
1. Open the required template folder
2. Download the following files:
   - `index.html`
   - `login.html`
   - `style.css`
3. Keep all files in **one folder**
4. Open `index.html`

---

## Folder Structure

Each template is independent and structured as follows:
